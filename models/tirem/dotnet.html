Terrain Integrated Rough-Earth Model (TIREM) - C#/.NET

Overview

Getting Started

To begin, install the NuGet package, TIREM, from the NTIA NuGet repository

All of the below code snippets requires the including the following using statement.

using ITS.Propagation;

Documentation

The tirem library is packaged to support IntelliSense when developing in Visual Studio. This documentation will provide the developer with information on function descriptions, variable details, etc.

Examples

Note: These examples use an artificial terrain profile.

Basic Transmission Loss

var heights = new[] { 4.5, 7, 3.6, 8, 10, 5 };
var dists = new[] { 0.0, 10000, 20000, 30000, 70000, 200000 };

double h_tx__meter = 5;
double h_rx__meter = 6;
double f__mhz = 1755;
double N_s = 301;
double sigma = 0.005;
double epsilon = 15;
double humid = 50;

TIREM.Invoke(h_tx__meter, h_rx__meter, f__mhz, heights, dists, 
    N_s, sigma, epsilon, humid, TIREM.Polarization.Vertical,
    out double A__db, out double A_fs__db, 
    out TIREM.TiremPropagationMode mode);

// A__db = 204.60               // basic transmission loss, in dB
// A_fs__db = 143.35            // free space loss, in dB
// mode = Troposcatter

// mode2 = Troposcatter

// tottro2 = 192.23             // diffraction loss, in dB
